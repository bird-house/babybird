Introduction
============

MyBird (the bird)
  *MyBird is just an minimal WPS Bird to get started ...*.

MyBird is a Python package with some proccess for  Web Processing Services (WPS). Currently it is using the `PyWPS <https://github.com/geopython/PyWPS>`_ server.

MyBird is part of the `Birdhouse`_ project.

.. _`Birdhouse`: http://bird-house.github.io/
